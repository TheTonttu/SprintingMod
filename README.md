# SprintingMod
Currently a Work In Progress.

## Install
Make sure to install SMAPI v0.37 or higher at https://github.com/ClxS/SMAPI/releases/tag/0.37 then download the SprintingMod.zip at https://github.com/oliverpl/SprintingMod/releases/tag/V1.0 and extract the contents into the Mods folder of your Stardew Valley game.
